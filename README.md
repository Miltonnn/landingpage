## 📖 Sobre o Projeto

Este projeto foi desenvolvido utilizando o **App Router do Next.js**, explorando recursos modernos do ecossistema React e priorizando organização, tipagem forte e componentização.

A aplicação foi construída com foco em:

- 📈 Performance
- 🧩 Componentização reutilizável
- 🔐 Segurança e previsibilidade com TypeScript
- 🎨 Consistência visual com shadcn/ui
- 🧼 Código limpo e escalável

---

### 🔎 Decisões Técnicas

- Uso de `_components` para evitar geração automática de rotas
- Separação entre Server e Client Components
- Uso da Metadata API para SEO
- Organização modular pensando em escalabilidade
